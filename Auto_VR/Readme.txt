# Udacity-VRND-Maze
Description: This project is a fully interactive VR experience to show how Virtual Rality is changing Automotive Industry.
Developer: Carla Julciane dos Santos
Environment: built to Android OS using Google VR SDK version 1.0.3 (developed and tested on Unity 5.6.2f1)



Main Challenges:
1-) Choose an industry where VR may have a great impact.
2-) Research on Automotive companies that already uses VR.
2-) Create a 360 video player.
4-) Create a show room where car rotates with the camera.
5-) Create a "cinema" room to show regular videos.
